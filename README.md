# 心理健康管理系统与APP设计

#### 介绍
心理健康管理系统，包含后台管理系统和用户APP。包含心理日记、心理论坛、心理咨询等必备功能，且增加最新调用AI大模型功能。可以做管理系统作业，也可做APP设计使用。

#### 软件架构

软件架构说明

1. Springboot框架
2. Java17
3. Mybatis
4. Maven
5. Mysql
6. 前后端分离
